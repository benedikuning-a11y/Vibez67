# Vibez Music Player 

## Features
- Login demo screen (no backend)
- Discover grid
- Playlist sidebar
- Now playing widget
- Play / Pause / Prev / Next
- Progress bar seek
- Volume slider
- Real audio playback using remote sample mp3 files

## How to run
1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

Open http://localhost:5173


You can replace the sample audio URLs in `src/App.jsx` with your own files.
